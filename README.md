# vectree
a n-dimensional tree of n-dimensional vectors that can be transformed, etc...

                                           (node_a)-->[a0,b0,...,n0]
                                          /  |  |  \
                                        /    |  |    \
                                      /      |  |      \
                                    /        |  |        \
                                  /          |  |          \
                                /            |  |            \
                              /              |  |              \
                            /                |  |                \
                          /                  |  |                  \
              (node_b)-->[a1,b1,...,n1]     (...)           (node_n)-->[an,bn,...,nn]
                  .                           .                .
                  .                           .                .
                  .                           .                .
